Understanding SQL DML Commands and Subqueries

What is DML?

Data Manipulation Language (DML) refers to the subset of SQL commands used to manipulate data in a database. The primary DML commands are:

  • INSERT: To add new records to a table.
  • UPDATE: To modify existing records.
  • DELETE: To remove records from a table.

Example of an INSERT command


INSERT INTO employees (id, name, position)
VALUES (1, 'John Doe', 'Software Engineer');

Example of an UPDATE command


UPDATE employees
SET position = 'Senior Software Engineer'
WHERE id = 1;

Example of a DELETE command


DELETE FROM employees
WHERE id = 1;

What are Subqueries?

A subquery, also known as a nested query, is a query within another SQL query. Subqueries can be used in SELECT, INSERT, UPDATE, or DELETE statements to perform complex filtering or transformations.

Example of a Subquery in a SELECT statement


SELECT * 
FROM employees 
WHERE id IN (
    SELECT manager_id 
    FROM managers
);

Example of a Subquery in an UPDATE statement


UPDATE employees
SET salary = salary * 1.1
WHERE id IN (
    SELECT id
    FROM high_performers
);

Example of a Subquery in a DELETE statement


DELETE FROM employees
WHERE department_id IN (
    SELECT id
    FROM departments
    WHERE name = 'Closed Department'
);

Subqueries are powerful tools in SQL that can be used to simplify complex queries or to build flexible query logic.
