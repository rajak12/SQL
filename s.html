Data Manipulation Language (DML) Commands

Data Manipulation Language (DML) commands are used to manage and manipulate data in SQL databases. These commands include:

  • INSERT: Insert new records into a table.
  • UPDATE: Update existing records in a table.
  • DELETE: Delete records from a table.
  • SELECT: Retrieve data from a table.

Subqueries in SQL

Subqueries, also known as inner queries or nested queries, are SQL queries embedded within another query. They allow you to perform operations that depend on the results of another query.

Types of Subqueries

  • Single-row subqueries: Return a single result, used with comparison operators like =, <, or>.
  • Multiple-row subqueries: Return multiple results, used with operators like IN, ANY, or ALL.
  • Correlated subqueries: Refer to outer queries, making them dependent on the outer query.

Examples of Subqueries

Here are some examples of using subqueries in SQL:


-- Find employees with salaries above the average
SELECT * FROM employees 
WHERE salary > 
  (SELECT AVG(salary) FROM employees);

-- Find products with prices higher than the average in a specific category
SELECT * FROM products 
WHERE price > 
  (SELECT AVG(price) FROM products WHERE category = 'Electronics');
