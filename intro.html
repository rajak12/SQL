Data Manipulation Language (DML) Commands

DML commands are used for managing data within tables. Common DML commands include:

  • INSERT - Adds new rows to a table
  • UPDATE - Modifies existing rows
  • DELETE - Removes rows from a table

Data Definition Language (DDL) Commands

DDL commands are used to define the structure of the database. Common DDL commands include:

  • CREATE - Creates a new table or database
  • ALTER - Modifies existing tables
  • DROP - Deletes tables or databases

Data Control Language (DCL) Commands

DCL commands control access to the database. Common DCL commands include:

  • GRANT - Grants permissions to users
  • REVOKE - Removes permissions from users

Querying Data Commands

SQL queries allow you to retrieve and manipulate data. The most common command is SELECT, which has various clauses like:

  • SELECT - Retrieves data from tables
  • WHERE - Filters results based on conditions
  • GROUP BY - Groups results based on specific columns
  • ORDER BY - Sorts results in ascending or descending order

Joining Commands

Joining commands allow you to combine data from different tables. Common join types include:

  • INNER JOIN - Combines rows with matching values
  • LEFT JOIN - Combines all rows from the left table and matching rows from the right
  • RIGHT JOIN - Combines all rows from the right table and matching rows from the left
  • FULL JOIN - Combines all rows from both tables

Subqueries in SQL

Subqueries are queries nested within other queries. They can be used in:

  • SELECT statements
  • WHERE clauses
  • HAVING clauses

Aggregate Functions Commands

Aggregate functions perform calculations on sets of data. Common aggregate functions include:

  • SUM - Adds values together
  • AVG - Computes the average
  • MIN - Finds the minimum value
  • MAX - Finds the maximum value
  • COUNT - Counts the number of rows
