Python Variables and Data Types

In Python, variables are used to store data. You can create a variable by assigning it a value using the `=` operator.

Python has several basic data types:

  • Integer: Whole numbers, such as 1, 2, or 3.
  • Float: Decimal numbers, such as 3.14 or 2.71.
  • String: Text, like 'Hello, World!'.
  • Boolean: Represents true or false values.

Example Code for Variables and Data Types

                    
    # Creating variables
    x = 10  # Integer
    y = 3.14  # Float
    greeting = "Hello, World!"  # String
    is_active = True  # Boolean
    
    # Printing variables
    print(x)  # Output: 10
    print(y)  # Output: 3.14
    print(greeting)  # Output: Hello, World!
    print(is_active)  # Output: True

Python Control Flow

Control flow allows you to make decisions and repeat actions in Python. The common control flow structures are:

  • If Statements: Make decisions based on conditions.
  • For Loops: Repeat actions over a sequence.
  • While Loops: Repeat actions while a condition is true.

Example Code for Control Flow

                    
    # If statements
    x = 10
    if x > 5:
        print("x is greater than 5")  # Output: x is greater than 5
    
    # For loops
    numbers = [1, 2, 3, 4, 5]
    for number in numbers:
        print(number)  # Output: 1 2 3 4 5
    
    # While loops
    count = 0
    while count < 3:
        print(count)  # Output: 0 1 2
        count += 1

Python Functions

Functions in Python allow you to encapsulate a block of code and reuse it. A function can take parameters and return a value.

Example Code for Functions

                    
    # Defining a function
    def add(a, b):
        return a + b
    
    # Using the function
    result = add(2, 3)  # Output: 5
    print(result)  # Output: 5
    
    # Function with default parameter
    def greet(name="World"):
        return f"Hello, {name}!"
    
    print(greet())  # Output: Hello, World!
    print(greet("Alice"))  # Output: Hello, Alice!

Python Lists and Dictionaries

Lists and dictionaries are common data structures in Python.

  • Lists: Ordered collections of items.
  • Dictionaries: Key-value pairs.

Example Code for Lists and Dictionaries

                    
    # Creating a list
    fruits = ["apple", "banana", "orange"]
    
    # Accessing list items
    print(fruits[0])  # Output: apple
    
    # Creating a dictionary
    person = {"name": "Alice", "age": 30}
    
    # Accessing dictionary values
    print(person["name"])  # Output: Alice
