Introduction to NumPy

NumPy is the core library for scientific computing in Python, providing a high-performance multidimensional array object and a variety of tools for manipulating these arrays. If you're into data science, machine learning, or analytics, understanding NumPy is essential.

In this blog, we'll cover the basics of NumPy, including array creation, indexing, slicing, and basic operations. Let's get started!

Creating Arrays

NumPy allows you to create arrays in various ways. Here's a look at some common techniques:

  • 1D Arrays: Create a simple one-dimensional array:
                    
    import numpy as np
    
    # Creating a 1D array
    a = np.array([1, 2, 3])
    print(a)
  • 2D Arrays: Create a two-dimensional array from a list of tuples:
                    
    # Creating a 2D array
    b = np.array([(1.5, 2, 3), (4, 5, 6)], dtype=float)
    print(b)
  • 3D Arrays: Create a three-dimensional array from nested lists:
                    
    # Creating a 3D array
    c = np.array([[(1.5, 2, 3), (4, 5, 6)], [(3, 2, 1), (4, 5, 6)]], dtype=float)
    print(c)
  • Placeholders: NumPy has functions to create placeholder arrays, such as zeros, ones, and identity matrices:
                    
    # Creating an array of zeros
    zeros = np.zeros((3, 4))
    
    # Creating an identity matrix
    eye = np.eye(2)
    
    print(zeros, eye)

Indexing, Slicing, and Subsetting

NumPy allows you to access and manipulate array elements with powerful indexing and slicing techniques:

  • Indexing: Select specific elements from arrays using simple indices or a combination of row/column indices:
                    
    # Select the element at the 2nd index
    print(a[2])
    
    # Select the element at row 1, column 2
    print(b[1, 2])
  • Slicing: Extract a range of elements from an array:
                    
    # Select items at index 0 and 1
    print(a[0:2])
    
    # Select items at rows 0 and 1 in column 1
    print(b[0:2, 1])
  • Boolean Indexing: Use conditions to filter array elements:
                    
    # Select elements from 'a' that are less than 2
    print(a[a < 2])

Arithmetic Operations

NumPy supports various arithmetic operations, allowing you to perform element-wise calculations, matrix multiplications, and more. Here are some examples:

  • Element-wise Operations: Perform basic arithmetic operations element-wise:
                    
    # Addition
    print(b + a)
    
    # Subtraction
    print(a - b)
    
    # Multiplication
    print(a * b)
    
    # Division
    print(a / b)
  • Aggregate Functions: NumPy has several aggregate functions for data analysis: 
                    
    # Calculate the sum of 'a'
    print(np.sum(a))
    
    # Get the mean of 'b'
    print(np.mean(b))

Array Manipulation

NumPy provides functions for reshaping, transposing, and modifying arrays. Let's explore some common array manipulation techniques:

  • Reshaping: Reshape arrays without changing data:
                    
    # Flatten the array
    print(b.ravel())
    
    # Reshape an array
    g = b.reshape(3, -2)
    print(g)
  • Adding/Removing Elements: Append, insert, or delete elements in an array:
                    
    # Append an element to 'a'
    print(np.append(a, 4))
    
    # Insert an element in 'a'
    print(np.insert(a, 1, 5))
    
    # Delete an element from 'a'
    print(np.delete(a, [1]))

Conclusion

NumPy is a versatile library that provides a solid foundation for scientific computing in Python. This blog covered the basics of array creation, indexing, slicing, arithmetic operations, and array manipulation. With this knowledge, you're ready to dive deeper into data science and explore more advanced topics with NumPy.
