What is SQL?

Structured Query Language (SQL) is the standard language for managing and manipulating databases. It is used to create, read, update, and delete data in relational databases.

SQL Commands

SQL has different types of commands to perform various operations on databases. The main types are:

  • Data Definition Language (DDL): Commands like CREATE, ALTER, and DROP, used to define the structure of a database.
  • Data Manipulation Language (DML): Commands like SELECT, INSERT, UPDATE, and DELETE, used to manipulate data.
  • Data Control Language (DCL): Commands like GRANT and REVOKE, used to manage user permissions.

Simple SQL Example

Here is a simple SQL query to select all records from a table called "employees":

                
    SELECT * FROM employees;
