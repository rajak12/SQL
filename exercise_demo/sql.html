Introduction to SQL

SQL (Structured Query Language) is a standard language for accessing and managing databases. It is widely used in data-driven environments to retrieve, insert, update, and delete data. In this blog, we'll explore SQL basics through practical examples and exercises.

Let's start with setting up a sample database for our SQL practice demo:

                
-- Create a sample table for practice
CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT,
    department VARCHAR(50),
    salary FLOAT
);

-- Insert sample data into the table
INSERT INTO employees (id, name, age, department, salary)
VALUES
    (1, 'Alice', 28, 'Engineering', 60000),
    (2, 'Bob', 35, 'Marketing', 55000),
    (3, 'Charlie', 32, 'Engineering', 70000),
    (4, 'Diana', 27, 'HR', 45000);

Basic SQL Operations

Now that we have a sample table, let's explore some basic SQL operations to interact with the data:

  • SELECT: Retrieve data from the table. Here's an example of selecting all records from the "employees" table:
                
-- Select all records from the employees table
SELECT * FROM employees;
  • WHERE: Filter data based on conditions. Here's an example of selecting employees from the "Engineering" department:
                
-- Select employees from the Engineering department
SELECT * FROM employees WHERE department = 'Engineering';
  • ORDER BY: Sort data in ascending or descending order. This example sorts employees by salary in descending order:
                
-- Sort employees by salary in descending order
SELECT * FROM employees ORDER BY salary DESC;
  • LIMIT: Limit the number of records returned by a query. This example retrieves the top 2 highest-paid employees:
                
-- Get the top 2 highest-paid employees
SELECT * FROM employees ORDER BY salary DESC LIMIT 2;

Intermediate SQL Operations

SQL also supports advanced operations like joins, aggregation, and subqueries. Let's explore these concepts with examples:

  • JOIN: Combine data from multiple tables. This example demonstrates an INNER JOIN between "employees" and another table, "departments":
                
-- Create a sample departments table
CREATE TABLE departments (
    department VARCHAR(50),
    location VARCHAR(50)
);

-- Insert data into departments
INSERT INTO departments (department, location)
VALUES ('Engineering', 'Building A'),
       ('Marketing', 'Building B'),
       ('HR', 'Building C');

-- Perform an INNER JOIN between employees and departments
SELECT employees.name, employees.department, departments.location
FROM employees
INNER JOIN departments
ON employees.department = departments.department;
  • Aggregation Functions: Use functions like COUNT, SUM, and AVG to analyze data. This example calculates the average salary in the "Engineering" department:
                
-- Calculate the average salary in the Engineering department
SELECT AVG(salary) AS average_salary
FROM employees
WHERE department = 'Engineering';
  • Subqueries: A query within another query. This example uses a subquery to find the names of employees who earn above the average salary:
                
-- Find employees who earn above the average salary
SELECT name, salary
FROM employees
WHERE salary > (
    SELECT AVG(salary)
    FROM employees
);
