SQL

SQL (Structured Query Language) is a standard language for managing databases. It is used to retrieve, insert, update, and delete data.

SQL Basics

Creating a Database

To create a database in SQL, use the CREATE DATABASE command.

CREATE DATABASE my_database;

Creating a Table

To create a table, specify the columns and their data types.


CREATE TABLE users (
    user_id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);

Basic SELECT Query

To select data from a table, use the SELECT command.

SELECT * FROM users;

Advanced SQL Topics

JOINS

JOINS allow you to combine data from multiple tables.


SELECT u.name, o.order_id 
FROM users u 
JOIN orders o ON u.user_id = o.user_id;

Aggregate Functions

Aggregate functions summarize data, like counting or averaging.

SELECT COUNT(*) FROM users;
