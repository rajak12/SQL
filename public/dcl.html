Data Manipulation Language (DML) Commands

Data Manipulation Language (DML) commands in SQL are used to manage and manipulate data in a database. Here are the common DML commands:

  • INSERT: Inserts new records into a table.
  • UPDATE: Updates existing records in a table.
  • DELETE: Deletes records from a table.
  • SELECT: Retrieves data from one or more tables.

Data Control Language (DCL) Commands

Data Control Language (DCL) commands in SQL are used to control access to data within a database. The primary DCL commands are:

  • GRANT: Grants permissions to users or roles.
  • REVOKE: Revokes permissions from users or roles.
