Data Manipulation Language (DML) Commands

Data Manipulation Language (DML) commands in SQL are used to manipulate the data in a database. These include:

  • INSERT - Adds new rows to a table.
  • UPDATE - Modifies existing rows in a table.
  • DELETE - Removes rows from a table.

Here's an example of an INSERT command:

INSERT INTO employees (id, name, department) VALUES (1, 'John Doe', 'Engineering');

An example of an UPDATE command:

UPDATE employees SET department = 'Marketing' WHERE id = 1;

And an example of a DELETE command:

DELETE FROM employees WHERE id = 1;

Data Definition Language (DDL) Commands

Data Definition Language (DDL) commands in SQL are used to define and manage the structure of a database. These include:

  • CREATE - Creates a new table or database.
  • ALTER - Modifies an existing table or database.
  • DROP - Deletes an existing table or database.

Here's an example of a CREATE TABLE command:

CREATE TABLE employees ( id INT PRIMARY KEY, name VARCHAR(50), department VARCHAR(50) );

An example of an ALTER TABLE command:

ALTER TABLE employees ADD COLUMN salary INT;

And an example of a DROP TABLE command:

DROP TABLE employees;
