How to install MYSQL workbench

Install MySQL Workbench

MySQL Workbench is the client we are going to use to connect to the MySQL server database.

To install MySQL Workbench please follow the instructions at On the download page, first select the operating system your computer uses (Windows, Mac, etc.) and then download and install MySQL Workbench Link

Note that the Login or Sign Up offered are optional and you don’t really need to login in order to download this app. On the lower part of the screen you’ll find a “No thanks, just start my download” link you should click.
