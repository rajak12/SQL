Introduction

Data Manipulation Language (DML) is a subset of SQL used for modifying data within a database. DML commands allow you to insert new data, update existing data, and delete data. Additionally, you can query or retrieve data from a database.

DML Commands

The four primary DML commands are:

  • INSERT
  • UPDATE
  • DELETE
  • SELECT

INSERT Command

The INSERT command is used to add new rows to a table. Here's an example: 

INSERT INTO employees (name, age, position) VALUES ('John Doe', 30, 'Engineer');

This example adds a new employee to the "employees" table with the specified values.

UPDATE Command

The UPDATE command is used to modify existing data in a table. Example: 

UPDATE employees SET age = 31 WHERE name = 'John Doe';

This example updates the age of an employee named "John Doe" to 31.

DELETE Command

The DELETE command is used to remove rows from a table. Example: 

DELETE FROM employees WHERE name = 'John Doe';

This example deletes the employee named "John Doe" from the "employees" table.

SELECT Command

The SELECT command is used to query data from a table. Example: 

SELECT * FROM employees;

This example retrieves all rows from the "employees" table.
