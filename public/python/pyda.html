Getting Started with Pandas

Pandas is a cornerstone library for data analysis in Python. It allows you to work with data in a structured way, similar to a spreadsheet. Let's explore some basic Pandas operations.

Creating a DataFrame

A DataFrame is a tabular data structure with labeled rows and columns. Here's how to create one from a dictionary:

                
import pandas as pd

# Create a simple DataFrame from a dictionary
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}

df = pd.DataFrame(data)
print(df)

Basic Data Manipulation with Pandas

Once you have a DataFrame, you can perform various operations to manipulate the data. Let's look at some basic examples:

  • Selecting Columns: Retrieve specific columns from the DataFrame.
  • Filtering Rows: Apply conditions to filter rows based on certain criteria.
  • Adding/Removing Columns: Add new columns or remove existing ones.
                
# Selecting specific columns
print(df[['Name', 'City']])

# Filtering rows where Age is greater than 30
print(df[df['Age'] > 30])

# Adding a new column
df['Salary'] = [50000, 60000, 70000]
print(df)

# Removing a column
df.drop('City', axis=1, inplace=True)
print(df)

Data Visualization with Matplotlib

Matplotlib allows you to create various types of plots and charts to visualize your data. Let's look at some simple examples of data visualization with Matplotlib.

Creating a Line Plot

A line plot is a simple way to visualize data over time or any continuous variable. Here's an example:

                
import matplotlib.pyplot as plt

# Data for the plot
x = [1, 2, 3, 4, 5]
y = [10, 20, 30, 40, 50]

# Create a line plot
plt.plot(x, y)
plt.xlabel('X Axis')
plt.ylabel('Y Axis')
plt.title('Simple Line Plot')
plt.show()

Creating a Bar Chart

A bar chart is useful for comparing different categories. Here's an example:

                
# Data for the bar chart
categories = ['A', 'B', 'C']
values = [5, 7, 3]

# Create a bar chart
plt.bar(categories, values)
plt.xlabel('Categories')
plt.ylabel('Values')
plt.title('Simple Bar Chart')
plt.show()
