Introduction to Pandas

Pandas is a popular Python library built on NumPy that provides easy-to-use data structures and data analysis tools. It is a cornerstone of Python-based data science and is widely used for data manipulation, analysis, and visualization.

If you're new to Pandas, the following sections will guide you through its basics and some advanced concepts.

Pandas Data Structures

Pandas provides two primary data structures for data analysis:

  • Series: A one-dimensional labeled array that can hold any data type. Here's an example of creating a Pandas Series:
                
import pandas as pd
s = pd.Series([3, -5, 7, 4], index=['a', 'b', 'c', 'd'])
print(s)
  • DataFrame: A two-dimensional data structure with labeled rows and columns. It can hold various data types across different columns. Here's an example of creating a DataFrame from a dictionary:
                
data = {
    'Country': ['Belgium', 'India', 'Brazil'],
    'Capital': ['Brussels', 'New Delhi', 'Bras√≠lia'],
    'Population': [11190846, 1303171035, 207847528]
}

df = pd.DataFrame(data)
print(df)

Basic Data Manipulation with Pandas

Pandas provides numerous functions for data manipulation and analysis. Let's explore some basic operations:

  • Selection: Selecting specific rows or columns from a DataFrame. You can use label-based selection with `loc` or position-based selection with `iloc`:
                
# Select a single row by label
df.loc[1]

# Select a single column by position
df.iloc[:, 1]

# Select multiple rows by label
df.loc[[1, 2]]
  • Filtering: Applying conditions to retrieve specific data. Here's an example of filtering rows where the population is greater than 200 million:
                
# Filter rows where population is over 200 million
filtered_df = df[df['Population'] > 200000000]
print(filtered_df)
  • Aggregation: Using functions like `sum`, `mean`, `median`, and `describe` to analyze data. Let's see how to calculate the sum of all populations:
                
# Calculate the sum of populations
total_population = df['Population'].sum()
print("Total Population:", total_population)

Data I/O with Pandas

Pandas makes it easy to read and write data from various sources. Here are some common operations for data I/O:

  • Read/Write to CSV: To read from a CSV file, you can use `pd.read_csv`. To write to a CSV file, use `df.to_csv`:
                
# Read from a CSV file
df = pd.read_csv('my_data.csv')

# Write to a CSV file
df.to_csv('output.csv')
  • Read/Write to Excel: Pandas supports reading and writing to Excel files. Here's an example:
                
# Write to an Excel file
df.to_excel('output.xlsx', sheet_name='Sheet1')

# Read from an Excel file
df = pd.read_excel('input.xlsx', sheet_name='Sheet1')
  • SQL Integration: Pandas can also interact with SQL databases. Here's an example of reading from and writing to a SQL database:
                
from sqlalchemy import create_engine

# Create a SQLite engine
engine = create_engine('sqlite:///:memory:')

# Write to a SQL table
df.to_sql('my_table', engine)

# Read from a SQL query
df = pd.read_sql("SELECT * FROM my_table;", engine)
