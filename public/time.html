Working with Dates in SQL

SQL provides various functions to manipulate and work with dates. Let's explore some common operations:

  • DATEPART: Extracts a part of a date, such as the year or month.
  • DATEADD: Adds a specified time interval to a date.
  • DATEDIFF: Computes the difference between two dates.
  • GETDATE: Returns the current system date and time.

Example Queries

                
-- Extracting the year from a date
SELECT DATEPART(YEAR, '2024-05-02') AS year;

-- Adding 10 days to a date
SELECT DATEADD(DAY, 10, '2024-05-02') AS new_date;

-- Calculating the difference in days between two dates
SELECT DATEDIFF(DAY, '2024-01-01', '2024-05-02') AS date_difference;

Working with Time in SQL

SQL also provides functions for handling time and timestamps. Here are some common operations:

  • TIMEPART: Extracts a part of a time, like hours or minutes.
  • CONVERT: Converts data between different types, useful for date and time formatting.
  • GETUTCDATE: Returns the current system UTC date and time.

Example Queries

                
-- Extracting the hour from a timestamp
SELECT DATEPART(HOUR, '2024-05-02 14:30:00') AS hour;

-- Converting a date to a specific format
SELECT CONVERT(VARCHAR, '2024-05-02', 103) AS formatted_date;

-- Getting the current UTC date and time
SELECT GETUTCDATE() AS utc_date_time;
